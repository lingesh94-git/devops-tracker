One-time Google Sheets setup
Follow these steps once. After this, every tick you make saves permanently to your Google Sheet — accessible from any device.
Step 1. Go to sheets.google.com → create a new sheet → name it DevOps Routine Tracker
Step 2. Copy the Sheet ID from the URL (the long string between /d/ and /edit)
Step 3. Go to console.cloud.google.com → New Project → name it anything
Step 4. Enable Google Sheets API for that project
Step 5. Go to IAM & Admin → Service Accounts → Create Service Account
Step 6. Create a JSON key → download it → open it → copy the full contents
Step 7. In your Google Sheet → Share → paste the client_email from the JSON → give Editor access

Private Key (private_key from JSON — paste the full key including BEGIN/END lines)
This info is stored only in this HTML file on your device. Nobody else can access it unless they have this file. Your other Google Drive files are completely safe — this service account only has access to the one sheet you shared with it.
